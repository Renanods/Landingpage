🐾 PetSys – Landing Page para Sistema de Petshops

Landing page desenvolvida para apresentar o PetSys, um sistema de gestão para petshops, com foco em organização, produtividade e aumento de faturamento.

📌 Sobre o Projeto

Esta landing page foi criada com o objetivo de divulgar o sistema PetSys, destacando seus benefícios e incentivando o visitante a entrar em contato ou solicitar uma demonstração.

A proposta principal da página é:

Resolver o problema da desorganização em petshops

Substituir papel e planilhas por um sistema automatizado

Reduzir faltas em agendamentos

Melhorar o controle financeiro

Aumentar a eficiência do negócio

🎯 Objetivo da Landing Page

Apresentar claramente o valor do sistema

Mostrar as principais funcionalidades

Gerar leads (contato / demonstração)

Transmitir profissionalismo e confiança

🖥️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando:

HTML5 – Estrutura da página

CSS3 – Estilização personalizada

Bootstrap – Layout responsivo e componentes visuais

A escolha dessas tecnologias permite:

Estrutura limpa e organizada

Desenvolvimento rápido e padronizado

Fácil manutenção

🧱 Estrutura da Página

A landing page é dividida em seções estratégicas:

🔹 1. Hero Section

Headline forte

Subtítulo explicativo

Chamada para ação (CTA)

🔹 2. Benefícios

Problemas que o sistema resolve

Destaque para organização e controle

🔹 3. Funcionalidades

Gestão de agendamentos

Controle financeiro

🔹 4. Seção Institucional

Missão ou propósito do sistema

🔹 5. CTA Final

Incentivo para contato ou demonstração
